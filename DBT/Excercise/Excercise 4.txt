2. Display the maximum Weight in the Parts table.
   select max(weight) Max_weight from P;

4. Display the total Quantity sold for part ‘P1’.
   select sum(qty) from SPJ where `P#` ='p1';

-------------------------------------------------------------------------------

5. Display the total Quantity sold for each part.
   select sum(qty) from SPJ group by `P#`;

-------------------------------------------------------------------------------

6. Display the average Quantity sold for each part.
   select avg(qty) from SPJ group by `P#`;

-------------------------------------------------------------------------------

7. Display the maximum Quantity sold for each part, provided the maximum
Quantity is greater than 800
select `P#`,max(qty)
 -> from SPJ
 -> group by `P#`
 -> having max(qty)>800;
    OR
    select `P#`,max(qty) Max_Qty from SPJ
 -> group by `P#`
 -> having Max_Qty>800;

9. Display the count of Projects going on in different cities.

10. What is the difference between COUNT(Status) and COUNT(*) ?
    Count(Status)- counts only valid status value
    Count(*) -counts all values including null values
